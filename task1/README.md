# Задание 1
## Текст задания
>Задание 1: Собери свой ValKey для production
> Представь, что в компании политика безопасности — все образы должны собираться из проверенных исходников на базе Red Hat Universal Base Image. Тебе нужно:
> 
> 1. Написать Dockerfile с multi-stage сборкой ValKey последней стабильной версии из исходников (с официального сайта ValKey.io)
> 2. Базовый образ для финального контейнера — Redhat ubi9/ubi-minimal
> 3. В контейнере должен быть только ValKey и его зависимости, никаких лишних пакетов
> 4. Конфигурация должна читаться из /etc/ValKey/ValKey.conf (файл лежит рядом с Dockerfile)
> 5. ValKey должен запускаться от непривилегированного пользователя (не root)
> 6. Проверить, что контейнер собирается и ValKey отвечает на PING

### Как собрать и проверить

```bash
# 1. Собрать и запустить образ
docker compose up -d

# 2. Проверить PING
docker exec task1-nexign-container-1 valkey-cli ping
